Visualization of Voltaire's library database


Project

This project allows you to statistically visualize the database of Voltaire's library. The visualization is based on three pillars:

  • Qualitative data: that correspond to categories like:
    • contributor name
    • contributor role
    • publication place
    • publication date
    • Volume number
  • Quantitative data: that corresponds to the number of books by each category mentioned above:
    • Number of books per contributor
    • Number of books by place of publication
    • Number of books by date of publication
    • Number of contributors per role
  • Chrononical pillar: to filter data between two dates

Type of chart

We have chosen six types of visualization that are the most readable and understandable:

  • Line
  • Bar
  • Area
  • Pie
  • Polar bar
  • Funnel

Usage

We choose a criterion, then we select the type of graph. Once the graph was displayed, a years slider appeared to choose a year interval. in addition, a multiple drop-down list containing data according to the chosen criterion is activated to add another filter.
